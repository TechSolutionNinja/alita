UI

UI

Alita 致力于把ReactNative运行在小程序平台，所以天然的支持ReactNative官方组件，除此之外，在用Alita 构建UI的还可以：

  1. 直接使用小程序内置组件&自定义组件， 复用小程序现有能力
  2. alita-ui，一套支持Android，IOS，微信小程序的三端UI组件库
